# Mercadito de Ropa

Proyecto simple para practicar publicación de anuncios usando HTML, CSS y JavaScript.

## Archivos

- `index.html` → Vista cliente (solo ver anuncios)
- `publicar.html` → Vista vendedor (publicar anuncios)
- `styles.css` → Estilos compartidos

## Notas

- Los anuncios se guardan **solo en memoria**, no se guardan al cerrar la página.
- Ideal para practicar antes de usar Firebase o una base de datos real.
